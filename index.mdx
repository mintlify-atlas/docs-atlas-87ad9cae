lazydocker is a terminal UI for Docker and Docker Compose that replaces a dozen scattered `docker` commands with a single, keyboard-driven interface. See all your containers, services, images, volumes, and networks at a glance — and act on them instantly.

## Key features

- **Real-time status** — see container state, health, and resource usage live
- **Log streaming** — follow logs per container or service, filtered by time
- **ASCII graphs** — visualize CPU and memory usage directly in the terminal
- **Attach & exec** — connect to running containers or open an interactive shell
- **Bulk operations** — up, down, stop, pull, or build all services at once
- **Custom commands** — extend lazydocker with your own Go-templated commands
- **Mouse support** — click to navigate panels alongside keyboard shortcuts
- **Image inspection** — browse ancestor layers and prune unused images
- **Configurable** — theme colors, border style, command templates, and more

<Note>
  lazydocker requires Docker >= 29.0.0 (API >= 1.24). Docker Compose >= 1.23.2 is optional but recommended for service management.
</Note>
